豊君の場合
 　公開　2018年12月12日
　知的障害に伴う運動発達遅滞
　２歳時からのLS-CC松葉杖訓練での取り組み
☆　生育歴
　在胎37週１日で帝王切開にて出産。上児が帝王切開だったので、次児帝王切開で出産する予定だったが、予定日よりも速く破水したために、早い出産となりました。
　出生体重2420g。
　市で行う乳児検診が４ヶ月に実施しているが、４ヶ月頃になっても首がすわってなく、身体も柔らかい感じだったので、乳児検診を市の療育センターで行いました。診察に当たってくださった先生より発育が遅いと言われ、再度受診と言われました。受診後、先生より知的障害と言われました。
　５ヶ月よりPT訓練開始。背臥位からの引き起こし・腹臥位からの頭部挙上。
　６ヶ月、首がすわり→寝返りの訓練・背臥位から側臥位となり座位への訓練・四つ這い訓練も行いました。
　１歳６ヶ月頃、持ち込みあぐら座位が可能→臥位から座位になれるようになりました。四つ這いの訓練と平行して、足の踏みしめ訓練。あぐら座でのずり這いを始めた頃、湯澤先生に出会いました。
↓・↓・↓・
 ∞　２歳３ヶ月時に、訓練会に参加されました。
　現症：臥位から座位まで可能。移動はいざり這い。立位・歩行不能。
　関節可動域：特に問題なし。しかし、身体を動かされたことが無いのか、身体の関節を動かされると、ほとんど全ての関節で、抵抗を示す。
　訓練内容：全身のストレッチ・四つ這い位・LS訓練・松葉杖訓練。
　LS１度。松葉進度１。SLB着用。
→　３ヶ月後
　時たまであるが、気分によって四つ這い移動を行う。LS２度となる。
→　４ヶ月後
　訓練の場で、四つ這いをよく行う。
　床からの立ち上がり訓練を開始する。
→　５ヶ月後
　松葉進度２となる。
→　７ヶ月後
　床からの立ち上がりで、足部に体重を載せ、上手く立ち上がるようになる。
→　９ヶ月後
　歩行器歩行を松葉杖訓練と並行して行う。SLB着用。
→　11ヶ月後
　歩行器歩行が、少し行えるようになる。
→　11ヶ月１週間後
　物に捕まり立ち上がるようになる。
　松葉杖歩行訓練を止める。歩行器歩行の方が、喜んで行うから・・・。
→　１年３ヶ月後
　片手介助での介助歩行が、少しできるようになる。SLB着用。
→　１年６ヶ月後
　物に捕まり立ち上がり、伝い歩きを始める。
→　２年１ヶ月後
　歩行器歩行を喜んで行い、屋内を自由に歩いている。
→　２年８ヶ月後
　階段昇降訓練に入る。階段昇降の訓練の方法は、片手で手摺りにつかまり、他の手を介助します。そして１段ずつ昇降します。これを左右共に行う。
　階段昇降訓練を嫌がり、行おうとしない。宥め宥めしながら行う。
→　２年８ヶ月２週間後
　持ち込み立位から、数歩の独歩を行う。SLB着用。
→　３年後
　持ち込み立位から、数メートルの独歩を行う。
　階段昇降を嫌がりながらも行うようになる。
→　３年１ヶ月後
　持ち込み立位から独歩をよく行うようになる。時々、後方にバランスを崩すことがあるが、尻から落ちている。
→　３年３ヶ月後
　室内の持ち込み立位から、素足で独歩を行うようになる。
　階段の昇りを行うようになる。喜んで行わないが、それ程の拒否もしない。
→　３年６ヶ月後
　階段の降りを行うようになる。
→４年１ヶ月後
　持ち込み立位から、よく独歩を行うようになる。だが、物に捕まり立ち上がり歩こうとはしない。
→４年８ヶ月２週間後
　床から一人で立ち上がる。裸足でもSLB着用でも同じように・・。独歩を良く行う。
↓・↓・↓・
 ◎　これまでの経過を振り返って・・・
　私と知り合う以前のPT訓練で、｢背臥位から側臥位となり座位への訓練｣が有ります。この子ども以外にも、これと同じ訓練を受けている子どもが多くいるのです。
　乳児の発達経緯を理解して指導しているのだろうか？
　乳児はどのような臥位をしていても、腹臥位となり→股関節と膝関節を曲げて尻を持ち上げるようにしながら→腕の力で体幹を持ち上げ→座位となります。この座位となる一巡の順の中に、四つ這い位が若干入っているのです。また、この順で座位となれば、四つ這い位に移行することが容易であって、四つ這い位から四つ這い移動へと進んでいくのです。
　ところが、「背臥位から側臥位となり座位への訓練」では、横座りとなる為に、四つ這い位になることが難しくなり、いざり這いを始めることとなるのです。
　運動発達遅滞児が、何故に難しい運動療法を受け続けなければならないのでしょうか？
　いずれここに記す機会を得て、この事同じような子どもたちを列記する機会が来るのではないかと悔やまれます。
　どうして、単純に遅れている運動機能を伸ばそうとしないのでしょうか？
　この子どもたちは、まひが有るわけではありません。運動機能を進歩させる能力が、どこかで食い違ってしまったに過ぎないのです。ですから、滞っている箇所から、手順を導いてあげれば改善進歩するのです。だが、誤指導によって誤学習を植え付けてしまっている為に、誤学習の訂正が必要となり、改善進歩に多くの時間を要することとなってしまうのです。
