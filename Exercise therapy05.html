筋力増強訓練 　　ゆざわひろみ
　　　　公開　2019年01月27日
　肢体不自由児の不自由な身体を少しでも改善するためには、運動療法と言われる治療手技を施すこととなります。
　その中でも筋力増強訓練は、一定の筋力が伴わなければ身体を動かしたり、姿勢保持ができない為に行われる訓練です。
　しかし乳幼児の運動療法中では、特に筋力増強訓練を取り入れる必要はありません。
　運動機能訓練や歩行訓練を経る過程で、身体を動かしたり姿勢保持に必要な筋力は獲得できるからです。
　特に筋力強化を図る必要の有る子どもの疾患としては、二分脊椎症かも知れません。
→・→・→・
 　腹筋が弱いと称されて、腹筋の筋力強化を図る指導者がおられますが、寝返りや腹這い、座位や四つ這いができるならば、それで充分の腹筋を有しています。
　筋力が乏しいと想われるのは、階段昇降時です。
　階段昇降をどの様に指導するかによって、筋力を求めるか求めないかに分かれるかも知れませんが、LS-CC松葉杖訓練法での階段昇降には、ある程度の下肢筋が必要となります。
→・→・→・
 　階段の手すりを両手で持ち、階段を蟹歩きのように横に上り下りする方を指導する法もありますが、この上り下りではここに記すような筋力の必要はありません。
　また、この階段昇降法で学ぶことも間違えではありません。
　ただ、この先に、階段を正面を向いて昇り降りるすることがないだけです。
　LS-CC松葉杖訓練法での指導は、最初から片手で手すりをつかみ、他の手は介助して練習に入ります。
　練習に適した階段の高さは15cm前後で、幅は25cm程が適当かと…。
　高さがあまりにも低いと、足を上げる練習になりません。また幅があまりにも広いと、子どもの足では同じ段の中で足の位置を変えなければならなくなります。
　手すりの高さは、子どもが立って胸の高さ程度です。
・　階段昇降練習は、昇りから練習に入ります。
→　階段の下に位置します。
→　子どもの片手で手すりをつかみ、他の手は介助者が保持します。
　介助者は子どもの後方から介助します。手すりを持つ手と介護する手を持つことによって、子どもから不安感を取り除きます。
→　手すりを持った方の足を一段上の段に上げます。
　一時的に片足で立つこととなります。
→　段に上げた足でもって、身体を持ち上げるように下となっている足を持ち上げます。
→　同じ段に両足が揃います。
→　手すりを持った手を上に伸ばし、手すりを持ち替えます。
→　手すり側の足を上の段に上げます。
→　上記の繰り返しです。左右共に行います。
・　昇ることが少しでもできるようになったならば、降りる練習に入ります。
→　階段の上に位置します。
→　子どもの片手で手すりをつかみ、他の手は介助者が保持します。
　介助者は子どもの前方に位置し介助します。手すりを持つ手と介護する手を持つことによって、子どもから不安感を取り除きます。
→　手すりを持った反対側の足を一段下の段に降ろします。
　一時的に片足で立つこととなります。
→　段に降ろした足でもって、身体を一時的に指示します。
→　同じ段に両足が揃います。
→　手すりを持った手を下に伸ばし、手すりを持ち替えます。
→　手すり側と反対の足を降ろします。
→　上記の繰り返しです。左右共に行います。
　これが階段昇降練習の開始ですが、昇りで片足で身体を押し上げるには、それなりの筋力が必要となります。
　階段昇降練習は、杖歩行や歩行器歩行が少しできるようになってから開始しますが、歩行練習の最初はフラットな床での練習ですから、それ程に筋力が求められません。階段昇降が上達することによって、独歩に必要な筋力となっていくのです。
